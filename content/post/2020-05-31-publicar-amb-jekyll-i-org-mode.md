Finalment he descobert una manera d’usar el meu estimat _emacs_ i el seu mode _orgmode_

Res més senzill que afegir al **\_config.yaml** el següent codi:

plugins: \[jekyll-org\]

Per instaŀlar en local les _gemes_:

sudo gem install jekyll-org

font: [jekyll-org](https://github.com/eggcaker/jekyll-org)

Nota: El problema és que a la pàgina github no es veu, en canvi en local funciona perfectament. Així, que de moment, seguiré investigant i escrivint en _markdown_.
